# 🏨 Hotel Management System (Java)

This project is a **console-based Hotel Management System** developed in **Java** using core **Object-Oriented Programming (OOP)** concepts and the **Java Collections Framework**. It is designed to manage hotel data, perform sorting operations, filter hotels by location, and handle basic user booking information.

## 🚀 Features
- Store and manage hotel details such as name, location, rating, price, and room availability  
- Sort hotels dynamically by:
  - Name  
  - Rating  
  - Room Availability  
- Filter hotels based on a specific city (e.g., Bangalore)  
- Manage user booking data including user ID, name, and booking cost  
- Display structured output for both hotel and user information  

## 🛠️ Technologies Used
- Java (Core Java)  
- OOP Concepts (Classes, Objects, Encapsulation, Comparable Interface)  
- Java Collections (ArrayList, Collections.sort)  

## 📂 Project Structure
- `Hotel` class: Stores hotel attributes and implements sorting logic using the `Comparable` interface  
- `User` class: Handles user details and booking cost  
- `Main` class: Contains the main logic for data initialization, sorting, filtering, and displaying results  

## ⚙️ Working
The program initializes hotel and user data using arrays, converts them into objects, and performs multiple operations such as sorting hotels based on different parameters, filtering hotels by location, and mapping users to their booked hotels. Sorting behavior is controlled dynamically using a static variable in the `Hotel` class.

## 🎯 Objective
To demonstrate how real-world hotel management functionalities can be implemented using Java programming concepts like sorting, object handling, and data processing in a simple and efficient way.

## ▶️ How to Run
1. Clone the repository  
2. Open in any Java IDE (Eclipse / IntelliJ)  
3. Compile and run `Main.java`
